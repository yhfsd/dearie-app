# 💜 Dearie
팬 활동 기록 및 감정 기반 인터랙션 웹 애플리케이션

Dearie는 사용자의 감정과 팬 활동을 기록하고,  
선택 기반 인터랙션을 통해 사용자 경험을 강화한 웹 애플리케이션입니다.

# 👨‍💻 프로젝트에서의 역할

### 🎯 Frontend

- React 기반 SPA 구조 설계 및 컴포넌트 아키텍처 구현  
- React Router를 활용한 페이지 라우팅 및 네비게이션 구조 설계  
- Zustand 기반 전역 상태 관리 구현 (챗봇 대화 및 사용자 데이터 관리)  
- localStorage를 활용한 사용자 데이터 및 콘텐츠 상태 유지 기능 구현  

---

### 🔗 API & 데이터 처리

- Axios 기반 REST API 연동 및 비동기 데이터 처리  
- Express API 서버와 통신하여 사용자 및 알림 데이터 관리  
- TypeScript를 활용한 데이터 타입 정의 및 상태 구조 설계  

---

### 🎨 UI/UX

- CSS3 / Flexbox / Grid 기반 반응형 UI 구현  
- Swiper.js를 활용한 동적 콘텐츠 슬라이드 구현  

---

### ⚙️ Backend & Deployment

- Express.js 기반 API 서버 구축  
- MongoDB Atlas 데이터베이스 연동  
- Vercel / Render를 활용한 배포 및 운영 환경 구축
