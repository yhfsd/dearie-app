[주요 업무]
	프로트엔드 아키텍처 설계 및 구현
	상태 관리 및 데이터 흐름 최적화
	UX/UI 구현 및 애니메이션 작업
	배포 및 운영 환경 구축


[활용 기술]
1.	프론트 엔드 개발
React.js -> 컨포넌트 기반 SPA 
설계, 타입 안정성 확보

React Router -> 페이지 라우팅 및 네비게이션

TypeScript -> 챗봇 기능 구현 
Zustand로 대화기록 및 사용자 선택 유지
대화 및 감정, 음악 데이터 등의 타입 정의
로컬 데이터를 통한 음악 데이터 제공

Axios -> REST API 연동 및 비동기데이터 처리

CSS3/FlexBox/Grid -> 반응형 레이아웃 구현

2.	데이터 흐름 최적화
LocalStorage -> 글등록, 사용자데이터 등록

3.	UXUI 구현 및 애니메이션 작업
Custom CSS 컴포넌트 -> 아코디언, 프로필 UI, 카드형 레이아웃

Swiper.js -> 반응형 슬라이드 제작
LocalStorage 데이터와 연동하여 동적 컨텐츠 슬라이드 구현

4.	배포 및 운영 환경 구축
GitHub -> 버전 관리 및 협업
Git Flow 전략 (Feature -> Develop -> Main)

Vercel -> 프론트엔드 배포
GitHub 저장소와 연동하여 main 브랜치 푸시 자동 빌드,배포 적용
